┌─ HOW IT WORKS ──────────────────────────────────────────────────────────┐
│                                                                         │
│  1. User visits page with tracking:                                     │
│     useSessionTracking({ platform: 'hub', userId })                    │
│                                                                         │
│  2. Hook automatically starts session:                                  │
│     POST /api/analytics/session { action: "start" }                    │
│                                                                         │
│  3. User browses for N minutes...                                      │
│                                                                         │
│  4. User closes tab / navigates away:                                  │
│     beforeunload / visibilitychange event fires                        │
│                                                                         │
│  5. Hook automatically ends session:                                    │
│     POST /api/analytics/session { action: "end" }                      │
│                                                                         │
│  6. Duration calculated and stored:                                     │
│     duration = endTime - startTime                                      │
│                                                                         │
│  7. Clerk metadata updated with:                                        │
│     - New session record                                                │
│     - Updated analytics summary                                         │
│     - Activity event logged                                             │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─ API ENDPOINTS ─────────────────────────────────────────────────────────┐
│                                                                         │
│  POST /api/analytics/session                                            │
│  └─ Start/end user session                                              │
│     Request: { action: "start"|"end", platform, sessionId }            │
│     Response: { sessionId, duration } or { error }                      │
│                                                                         │
│  GET /api/analytics                                                     │
│  └─ Get user analytics data                                             │
│     Query: ?type=summary|sessions|events&limit=20                      │
│     Response: { analytics } or { sessions } or { events }              │
│                                                                         │
│  GET /api/analytics/dashboard                                           │
│  └─ Get admin aggregated data (admin only)                              │
│     Response: { summary, userAnalytics }                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
